# Notes Studier

An application that allows you to upload your handwritten notes, then generate summaries, quizzes and tests based on your noes.

## Overview

This project is designed for students and lifelong learners who want to make the most out of their handwritten notes.
By leveraging AI, the application helps users to quickly generate summaries, quizzes, and tests from your notes, enhancing your study efficiency and retention of information.

## Tech Stack

- **Frontend:** React, Tailwind
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Rate Limiter:** Upstash
- **Hoster:** Render.com

## Features

- Upload handwritten notes (PDF)
- Generating summaries of notes
- Generating quizzes based on notes
